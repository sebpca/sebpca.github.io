InterPro - EMBL-EBI / Front-end web developer, designer

Redesign, develop and maintain the InterPro website (Protein sequence analysis & classification), one of the top resources in EBI, using in its last version, a template based on Foundation. First time styling in React, a JavaScript framework allowing us to build rich web interfaces. Particular attention to design clean and attractive homepage, browse section and entry pages. Use icon fonts and produce scalable SVG images (icons, banner, illustrations, etc.), to make the website look nice, modern and responsive on all different sizes of devices. Create and update the InterPro logo. Collaborate in the redesign of the main EBI template. Apply Agile/Scrum method on a daily base, being part of a multidisciplinary team (bioinformatician, back-end and UX experts).
